# IITK Authentication in Haskell

Add your username password as stated in line 65 eg
```haskell
parse [] = return ("username","password")
```
comment the line
```haskell
-- parse _ = readInput
```

Install using 
```sh
cabal install
```

then simply run 

```sh
hwall-auth-iitk
```
